# [P3] ncPeopleGrid.vue uses explicit component import while Nuxt auto-resolves components

## File
components/ncPeopleGrid.vue

## Issue
Line 14: `import ncBioCard from './ncBioCard.vue'` explicitly imports the component. Nuxt auto-resolves components in the `components/` directory, so this import is unnecessary. The original `ncJudgesGrid.vue` also had this import (before the PR), so it was carried forward, but it could be cleaned up.

## Suggested Fix
Remove `import ncBioCard from './ncBioCard.vue'` and let Nuxt auto-resolve the component. Use `<nc-bio-card>` (kebab-case) in the template if not already.
