# [P3] resources/index.vue has `.tagline` class in template but no corresponding CSS rule

## File
pages/resources/index.vue

## Issue
Line 4 uses `<p class="tagline">Explore resources from the New Commons initiative.</p>` but the `<style>` block has no `.tagline` rule. The paragraph will render with default styles, which may or may not be the intent.

## Suggested Fix
Add a `.tagline` CSS rule in the scoped style block to style the tagline appropriately, or remove the class if default styling is acceptable.
