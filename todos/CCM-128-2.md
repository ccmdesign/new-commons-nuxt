# [P1] ncJudgesGrid and ncObserversGrid are now dead code -- not deleted after ncPeopleGrid replacement

## File
components/ncJudgesGrid.vue, components/ncObserversGrid.vue

## Issue
The PR introduces `ncPeopleGrid.vue` as a generalized replacement and updates all usages in `the-2025-challenge.vue` and `indigenous-languages.vue` to use `nc-people-grid`. However, the original `ncJudgesGrid.vue` and `ncObserversGrid.vue` files remain in the codebase untouched. They still hardcode `useJudges` / `useObservers` imports. No remaining template references these components (grep confirms zero usages). They are dead code that will confuse future developers about which component to use.

## Suggested Fix
Delete `components/ncJudgesGrid.vue` and `components/ncObserversGrid.vue` since they have been fully replaced by `ncPeopleGrid.vue`.
