# [P3] `the-2025-challenge.vue` hero image alt text says "Hero Image" -- not descriptive

## File
pages/index.vue

## Issue
Line 25 (after the PR changes): `<img src="/assets/patterns/hero.jpg" alt="Hero Image" />`. The alt text "Hero Image" is not descriptive -- it describes the element type rather than the content. This is an accessibility issue as screen readers will announce "Hero Image" which is not meaningful.

## Suggested Fix
Replace `alt="Hero Image"` with a descriptive alt like `alt="Abstract pattern representing data commons"` or an appropriate description of what the hero image actually shows.
