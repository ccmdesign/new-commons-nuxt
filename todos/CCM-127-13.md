# [P3] Inline styles used in index.vue and incubator/2026.vue templates

## File
pages/index.vue, pages/incubator/2026.vue

## Issue
Several elements use inline `style` attributes:
- `index.vue`: `<div style="flex: 3;">`, `<NuxtLink style="font-size: var(--size-0); font-weight: 300; text-decoration: underline;">`, `<nc-button ... style="align-self: center;">`
- `incubator/2026.vue`: `<div style="flex: 3;">`, `<nc-button ... style="align-self: center;">`

These inline styles are harder to maintain and override than scoped CSS classes. They also pre-existed in the codebase for some (the `flex: 3` pattern), but the PR adds new inline styles for the NuxtLink.

## Suggested Fix
Extract inline styles into scoped CSS classes in the respective `<style>` blocks. For example, create a `.flex-grow-3` utility class and a `.challenge-link` class.
