# [P2] Initiatives data array duplicated across three files -- risks content drift

## File
pages/index.vue, pages/initiatives.vue

## Issue
The same initiatives data array (title, description, status, to, image for "The 2025 Challenge" and "Indigenous Languages") is duplicated verbatim in `pages/index.vue` (lines ~55-69) and `pages/initiatives.vue` (lines ~23-37). If one is updated and the other is not, the homepage and initiatives page will show different data. This violates DRY and is a maintenance risk.

## Suggested Fix
Extract the initiatives array into a composable (`composables/useInitiatives.js`) or a shared constant, then import it in both pages. Follow the existing pattern from `useJudges.js` / `useObservers.js`.
